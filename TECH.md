# Tech Stack

This web application is built using the following technologies:

## Frontend
- **Framework**: [Next.js](https://nextjs.org/) (Latest version, App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: React Server Components & Client Components

## Backend
- **Server**: Next.js Server Actions (Node.js runtime)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Database**: [SQLite](https://www.sqlite.org/) (Local file-based database)

## Tools & DevOps
- **Package Manager**: npm
- **Linting**: ESLint
